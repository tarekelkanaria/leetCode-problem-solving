# Add Binary

---

<font color="#1a237e">

Given two binary strings `a` and `b`, return _their sum as a binary string_.

</font>

## Example 1:

```
Input: a = "11", b = "1"
Output: "100"
```

## Example 2:

```
Input: a = "1010", b = "1011"
Output: "10101"
```

## About the solution

- **Time complexity :** `O(1)`

- **Space complexity :** `O(1)`
